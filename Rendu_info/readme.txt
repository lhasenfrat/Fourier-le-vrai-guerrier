FICHIERS .JAVA :
-Complex.java : Classe qui gère les objets de type complexe
-DispPan.java : Classe qui gère l’affichage
-main.java : la classe principale à exécuter.
-PanneauEntree.java : Classe gérant l’acquisition du dessin de l’utilisateur
-TemplateButton.java : Classe permettant de définir des boutons afin d’afficher des formes déjà paramétrées.
-FFT : Classe importée qui permet de calculer la transformée de Fourier d’un tableau de complexes
